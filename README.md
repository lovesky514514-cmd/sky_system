# SKY蓝天陪伴系统

一个面向长期陪伴场景的桌面端 AI 陪伴系统原型。  
系统结合用户注册登录、人格画像问卷、智能记忆精炼、记忆库管理与用户类型区分，构建一个更具连续性和个性化的陪伴式交互框架。

---

## 项目简介

SKY蓝天陪伴系统致力于解决传统聊天系统“只会回复、不会记住”的问题。  
本项目围绕以下几个核心方向展开：

- 长期记忆管理
- 人格画像生成
- 智能记忆精炼
- 普通用户 / VIP 用户分层逻辑
- 桌面端交互式客户端实现

当前版本为桌面端原型系统，支持本地运行，并可结合 DeepSeek API 实现智能语义精炼。

---

## 核心功能

### 1. 用户注册与登录

系统支持本地账号注册与登录，用户数据保存在本地 `users.json` 中。

### 2. 首次登录人格画像问卷

新用户首次登录时会自动弹出 15 题人格画像问卷。  
问卷完成后，系统自动计算五维人格结果：

- 开放性 O
- 尽责性 C
- 外向性 E
- 宜人性 A
- 神经质 N

并进一步生成：

- 人格分数
- 人格标签
- 人格总结

### 3. 智能聊天与记忆精炼

用户在聊天界面输入内容后，系统会：

- 显示本地陪伴回复
- 调用记忆精炼模块分析输入内容
- 判断内容是否值得记忆
- 提取关键词、记忆类型、重要度、建议层级
- 将满足条件的内容写入记忆库

### 4. 普通用户与 VIP 用户区分

系统支持两类用户：

- 普通用户
- VIP 用户

两类用户在记忆保存阈值与最近记忆显示数量上存在差异。

### 5. 记忆库管理

系统会将通过精炼后写入的内容写入本地 `memory.json`，并在界面中展示：

- 原始内容
- 精炼摘要
- 记忆类型
- 重要度分数
- 建议层级
- 关键词
- 精炼来源
